                                    # 🛡️ PhishShield – URL Risk Analyzer

## Overview
PhishShield is a Flask-based web application that analyzes URLs for phishing indicators using heuristic analysis and external security services.

## Features
- URL heuristic analysis
- Risk scoring
- VirusTotal integration
- WHOIS lookup
- SSL certificate inspection
- SQLite scan history
- Dashboard analytics
- PDF report generation

## Technologies
- Python
- Flask
- SQLite
- HTML/CSS
- Chart.js
- VirusTotal API

## Installation

git clone <https://github.com/Krishna-Gupta-Git/PhishShield-URL-Risk-Analyzer.git>

cd PhishShield-URL-Risk-Analyzer

pip install -r requirements.txt

python app.py

Visit:
http://127.0.0.1:5000
